My first blog post

This is the first blog post to be created on this site! Read here how I set up the site and got it up and running.

My first blog post. Now, I’m pretty new to this, so I’m not 100% sure how to write one of these. But here it goes.
A few weeks ago (from the time of writing this) I decided that I wanted to set up a website to document my journey and progress as a developer. I started making this website in HTML and originally just had it running on my local PC. I later set up an actual site and got it online. I used a site called Netlify to create it. It took some time to get the HTML code to run on it, but once it was running all I had to do was configure it.

I’m not really good at HTML and am quite new to it in general, so this has been a learning experience for me. Thankfully, my class is learning it at school at the moment in CAT (Computer Applications Sciences) so I am able to get a basic understanding of what is needed for the site to work and look good.

I guess I should start by telling you about me as a person.

My name is Thomas and I am a student from South Africa. I have been programming for a while now, but am far from anything impressive. I started off my journey with C# and Unity Game Engine. This is what I have the most knowledge on at the moment. From there, I made a few small games but never finished any. In school, I take the subjects IT, CAT and Art, so it is majority tech related. In IT, our curriculum states that we should learn Delphi, which is a programming language based off of Pascal. Originally, I thought that Delphi was useless and that I wouldn’t use it anywhere. But I have since realized that it has many uses and is mainly used as a learning language for students. As it was created Anders Hejlsberg (the same person who created the language C#) I have found it useful as it relates to what I have already learnt. This meant that a lot of the concepts that we have learnt in school, I already had prior knowledge on.

After this, I got interested in making mods for Minecraft. So, I started learning Java. Now this didn’t last very long as at the time I had a pretty weak PC and struggled to run the base game, let alone mods. However, this allowed me to gain an understanding of the language, which will benefit me in the long run. From there, I returned to C# and attempted to write a Chess AI. This, once again, didn’t last long as I did not have an understanding of how intricate these bots are. Maybe it can be a project that I can revisit some time?

I soon realized that C# was not the best language for me to do things in, as although it had a large amount of uses, it was not as powerful / robust as other languages. So, I decided to learn Python. This is the most recent thing that I have learnt and I have been working with Flask (A web development extension) that allows me to write APIs for different programs. I have recently started work on a mobile chatroom app that is being made in Unity C#, but uses an API that I wrote (and am hosting on my network) using Python. This has allowed to learn how Python works, how to use text-files and databases to store information, as well as how to use HTTP requests to send data to and from the server. It is not currently ported forward, as it is running on an old Android TV box running Linux. So, it doesn’t have much power to accommodate for many users at once. But as soon as I am able to, I will be publishing the app and making the server public for anyone to use.

That is basically all I have to say at this point, this post is mainly to help you, the reader, to get an understanding of what I have done up until this point in time. I will hopefully be making more posts as I want to keep the world updated on my progress as a developer. I will also be keeping this site updated with any projects that I may be working on in the future, and will soon have a section set up for people to contact me and suggest / assist me in my development.
